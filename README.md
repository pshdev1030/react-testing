## Types of Tests

### Unit tests

- 작은 단위의 코드를 고립된 환경에서 테스트

### Integration tests

- 두 개 이상의 단위의 코드를 같이 테스팅

### Functional Tests

- 함수의 특정 부분이 소프트웨어에서 잘 동작하는지 테스팅

### Ent-to-end (E2E) Tests

- 실제 브라우저와 서버 환경에서 테스트

## RTL

가상돔으로 테스팅, 브라우저 없이도 테스트 가능
